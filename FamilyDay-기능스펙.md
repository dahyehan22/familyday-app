# FamilyDay — 기능 스펙 문서

> 부모와 초등학생 자녀(6~12세)가 함께 사용하는 가족 일정/할일 관리 모바일 앱
> **타겟 디바이스**: 모바일 퍼스트 (iPhone 14 기준 390×844)

---

## 내비게이션 구조

```
하단 GNB (4탭, 스크롤 다운 시 숨김/업 시 표시)
├── 🏠 홈 — 오늘의 일정 + 할 일 보드 + 별 리워드 + 쿠폰 사용
├── 📅 캘린더 — 일간 / 주간 / 월간 탭 전환, 일정 추가/반복
├── 📋 타임라인 — {아이이름}의 하루 세로형 타임라인
└── 👥 가족 — 비로그인: 이메일+비밀번호 인증 / 로그인: 마이페이지 + 쿠폰 관리
```

---

## 구현 완료 기능

### 1. 홈 / 할 일 보드

| 항목 | 스펙 |
|------|------|
| 인사 헤더 | "안녕, {아이이름}네 가족! 👋" + 오늘 날짜 (한국어 포맷, `weekday:"long"` → "2026년 3월 15일 일요일") |
| 아바타 | 아이 얼굴 캐릭터 (`CuteFace`) |
| 오늘의 일정 카드 | 최대 3개 이벤트, 컬러 도트 표시 |
| 할 일 보드 | 👩 엄마 섹션 (핑크 태그) + 🧒 {아이이름} 섹션 (보라 태그), 전체/엄마/아이 필터 탭 |
| 할 일 상태 관리 | 일주일 단위 관리, 주 시작(월요일) 시 리스트 리셋. 미완료 할 일은 주 내에서 다음날로 자동 이월. 완료된 할 일은 완료한 날에만 표시 |
| 엄마 할 일 완료 | 체크박스 클릭으로 토글 |
| 아이 할 일 완료 | "완료 ✓" 버튼 → 체크 애니메이션 + 취소선 + ⭐ 별 파티클 폭발 |
| 할 일 추가 | 플로팅 "+ 할 일 추가" 버튼 → 바텀시트 모달 (담당자 선택, 텍스트 입력, 매주 반복 토글) |
| 매주 반복 | 할 일 추가 시 "매주 반복" 체크 → 새 주 시작 시 자동 재생성 |
| 할 일 수정 | 연필 아이콘 버튼 또는 텍스트 클릭 → 수정 모달 (텍스트, 담당자 변경) |
| 연속 달성 배너 | 아이 할일 2일 이상 연속 완료 시 홈에 "🔥 N일 연속 달성!" 배너 표시 |
| 별 현황 | "별 모으기" 카드 (모은 별 개수 표시, 최대값 없음) → 클릭 시 쿠폰 사용 페이지 이동 |
| 아이 이름 동적 적용 | `user.members`에서 `role:"아이"` 멤버 이름 추출, 헤더/할 일 보드/타임라인 등 전체 반영 |

### 2. 쿠폰 사용 (아이용)

| 항목 | 스펙 |
|------|------|
| 진입 | 홈 하단 "별 모으기" 카드 클릭 |
| 상단 현황 | 모은 별 개수 표시 (최대값 없음, 쿠폰 사용 시 차감 반영) |
| 쿠폰 리스트 | 별 갯수별 쿠폰 카드 (이모지, ★ 비용, 제목, 설명) |
| 사용 버튼 | 별 충분 → 초록 활성, 부족 → 회색 비활성 |
| 사용 확인 팝업 | 쿠폰 이모지 + 제목 + "★ N개의 별이 차감됩니다" + 취소/사용하기 |
| 사용 완료 | 버튼이 "사용 완료"로 전환, 별 차감 반영 |
| 빈 상태 | 쿠폰 없을 때 안내 메시지 |
| 뒤로가기 | 상단 뒤로가기 버튼 → 홈 복귀 |

### 3. 쿠폰 관리 (부모용)

| 항목 | 스펙 |
|------|------|
| 진입 | 가족 탭 > 마이페이지 > "쿠폰 관리" 카드 클릭 |
| 쿠폰 리스트 | 등록된 쿠폰 목록 (이모지, 별 비용, 제목, 설명) |
| 쿠폰 추가 | 상단 "+ 쿠폰 추가" 버튼 → 바텀시트 모달 |
| 추가 모달 | 이모지 선택 (16종), 별 갯수 입력, 쿠폰 이름, 설명(선택), 미리보기 |
| 쿠폰 수정 | 연필 아이콘 버튼 → 수정 모달 (이모지, 별 비용, 이름, 설명 편집) |
| 쿠폰 삭제 | 휴지통 버튼 → 확인 팝업 → 삭제 |
| 자동 정렬 | 추가 시 별 갯수 순 오름차순 정렬 |
| 데이터 연동 | `coupons` state로 관리, 부모가 추가/삭제하면 아이 쿠폰 사용 페이지에 즉시 반영 |

### 4. 주간 캘린더

| 항목 | 스펙 |
|------|------|
| 날짜 헤더 | 월~일, 오늘 보라색 원 하이라이트 |
| 주 이동 | `<` `>` 버튼 또는 좌우 터치 스와이프 |
| 시간 그리드 | 08:00~20:00, 1시간 단위 (44px 높이) |
| 이벤트 블록 | 색상 코딩, 시작~종료 시간 기반 높이 계산 |
| 이벤트 상세 | 이벤트 탭 → 바텀시트 |
| 전환 애니메이션 | 좌/우 슬라이드 |

### 5. 월간 캘린더

| 항목 | 스펙 |
|------|------|
| 헤더 | "2026년 3월" + `<` `>` 월 이동 |
| 달력 그리드 | 일/토 요일 색상 구분 (빨강/파랑) |
| 일정 표시 | 일정 있는 날에 컬러 도트 (최대 3개) |
| 오늘 표시 | 보라색 원 하이라이트 |
| 날짜 탭 | 바텀시트 → 해당 날 일정 목록 또는 "일정이 없어요" + 추가 버튼 |
| 다가오는 일정 | 하단 리스트 + "+ 추가" 버튼 |
| 일정 추가 모달 | 이름, 날짜 picker, 시작/종료 시간 picker, 16개 이모지, 8개 색상, 실시간 미리보기 |
| 데이터 연동 | 추가한 일정이 달력 도트 + 다가오는 일정 + 주간 캘린더에 즉시 반영 |

### 6. 타임라인 — {아이이름}의 하루

| 항목 | 스펙 |
|------|------|
| 헤더 카드 | 아바타 + "{아이이름}의 하루 ✨" + 완료 현황 (예: 5/11 완료) |
| 세로형 레이아웃 | 중앙 세로 스파인 + 좌우 교차 배치 |
| 스파인 | 세그먼트별 그라데이션 색상 전환, 하단 화살표 포인트 |
| 시간 리본 | 그라데이션 배경의 시간 뱃지 (한쪽에 배치) |
| 콘텐츠 카드 | 이모지 + 라벨 (반대쪽에 배치) |
| 일정 노드 (11개) | 기상 07:00 → 아침식사 → 등교 준비 → 학교 → 점심 → 숙제 → 피아노 → 놀이시간 → 저녁식사 → 자유시간 → 취침 21:00 |
| ✅ 완료 상태 | 세그먼트 색상 원 + 체크마크 + "완료 ✓" 라벨 |
| 🟣 진행중 상태 | 보라 글로우 원 + 이모지 + 이중 글로우 링 애니메이션 + 프로필 아바타(`CuteFace`) + 말풍선 "지금!" |
| ⚪ 미래 상태 | 연한 회색 원 + 흐릿한 이모지 |
| 자동 스크롤 | 진행중 노드로 자동 스크롤 |

### 7. 캘린더 (독립 페이지)

| 항목 | 스펙 |
|------|------|
| 헤더 | 일간 / 주간 / 월간 pill 탭 전환 |
| 일간 | 오늘의 일정 목록 (홈과 동일 데이터) |
| 주간 | 주간 캘린더 + 시간 그리드 (WeeklyCalendar) |
| 월간 | 월간 캘린더 + 다가오는 일정 + 일정 추가 (MonthlyCalendar) |
| 일정 반복 | 반복 안함 / 매일 / 매주 / 매월 |
| 매주 반복 | 복수 요일 선택 (월~일 칩) + 요일별 시작/종료 시간 개별 설정 + N주 반복 |
| 매일·매월 반복 | 동일 시간 × N회 반복 |
| 일정 추가 UI | 바텀시트 모달 (dimmed 오버레이 + bottom 정렬, 60vh) |

### 8. 가족 — 인증 (이메일+비밀번호)

| 항목 | 스펙 |
|------|------|
| 인증 방식 | 이메일 + 비밀번호 (Supabase Auth) |
| 회원가입 | 이메일 + 비밀번호(6자 이상) 입력 → Confirm email ON 시 인증 메일 발송 → 링크 클릭 후 로그인 가능 |
| 로그인 | 이메일 + 비밀번호 입력 → `signInWithPassword` |
| 로그인/회원가입 전환 | 상단 pill 탭으로 전환 |
| 에러 처리 | 중복 이메일, 잘못된 비밀번호, Rate limit(429) 대응 |
| 비밀번호 재설정 | "비밀번호를 잊으셨나요?" → 이메일 입력 → 재설정 링크 발송 → 링크 클릭 → 새 비밀번호 입력 UI (PASSWORD_RECOVERY 이벤트 감지) |
| 회원가입 시 초대 코드 | 회원가입 폼에 초대 코드(선택) 입력란 + 역할/아이콘 선택 → 가입 완료 시 해당 가족에 자동 합류 |
| 세션 유지 | `onAuthStateChange` 리스너로 자동 로그인 유지 |

### 9. 가족 — 마이페이지 (로그인 후)

| 항목 | 스펙 |
|------|------|
| 프로필 사진 | 클릭 → 이미지 업로드 (FileReader, 추후 Supabase Storage) |
| 닉네임 | 인라인 편집 (연필 아이콘 → 입력 → 저장) |
| 이메일 | 표시만 (수정 불가) |
| 가족 구성원 | 이름 + 역할 + 이모지 아바타 목록 |
| 구성원 추가 | "+ 구성원 추가" 버튼 → 바텀시트에서 이름/역할/이모지 입력 (프로필 구성원, 계정 없음) |
| 가족 초대 | "초대 코드 생성" → 6자리 코드 발급 (24시간 유효, 1회 사용) → 공유 → 상대방이 코드 입력하여 합류 |
| 초대 코드 합류 | "초대 코드로 가족 합류" → 코드 + 역할 + 아이콘 선택 → 기존 가족에서 나가고 새 가족에 합류 |
| 구성원 삭제 | 삭제 버튼 (본인 제외) |
| 쿠폰 관리 | "쿠폰 관리" 카드 → 쿠폰 관리 페이지 이동 |
| 로그아웃 | 하단 로그아웃 버튼 → `supabase.auth.signOut()` |

#### 구성원 유형

| 유형 | 설명 | 추가 방식 |
|------|------|----------|
| 연결된 구성원 | 실제 계정이 있는 가족 | 초대 코드로 합류 (마이페이지 또는 회원가입 시) |
| 프로필 구성원 | 계정 없는 가족 (어린 자녀 등) | 부모가 이름/역할 직접 등록 |

## 데이터 모델

### Event (일정)

| 필드 | 타입 | 설명 |
|------|------|------|
| id | string | uid()로 생성 |
| title | string | 예: "피아노 레슨" |
| date | string | "YYYY-MM-DD" |
| startHour | number | 0-23 |
| startMin | number | 0-59 |
| endHour | number | 0-23 |
| endMin | number | 0-59 |
| color | string | HEX 색상 |
| emoji | string | 이모지 아이콘 |

### TodoItem (할 일)

| 필드 | 타입 | 설명 |
|------|------|------|
| id | string | uid()로 생성 |
| text | string | 예: "책 읽기 30분" |
| owner | "mom" \| "kid" | 담당자 |
| isDone | boolean | 완료 여부 |
| starReward | number | 기본 1 |
| createdDate | string | 생성일 (YYYY-MM-DD) |
| isWeekly | boolean | 매주 반복 여부 |
| doneDate | string? | 완료한 날짜 (YYYY-MM-DD), 완료 시 기록 |

### Timeline Item (타임라인)

| 필드 | 타입 | 설명 |
|------|------|------|
| id | string | 식별자 |
| time | string | "HH:MM" |
| hour | number | 시간 |
| min | number | 분 |
| label | string | 예: "기상" |
| emoji | string | 이모지 아이콘 |

### Coupon (쿠폰)

| 필드 | 타입 | 설명 |
|------|------|------|
| id | string | uid()로 생성 |
| emoji | string | 이모지 아이콘 |
| starCost | number | 필요한 별 갯수 |
| title | string | 쿠폰 이름 (예: "간식 선택권") |
| desc | string | 설명 (선택) |

## 미구현 기능 (향후 개발)

> 상세 계획은 `FamilyDay-개발계획.md` 참고

### 🔐 Supabase 연동
- Magic Link 실제 인증 (`signInWithOtp`)
- DB 테이블 (profiles, families, events, todos, coupons, coupon_usage)
- RLS 기반 가족 단위 접근 제어
- 가족 초대 링크

### 🔔 쿠폰 사용 알림
- 아이가 쿠폰 사용 완료 시 부모에게 알림
- 푸시 알림 또는 인앱 알림

### 기타
- 일정 수정/삭제
- 프로필 사진 Supabase Storage 업로드
- 푸시 알림 (일정 리마인더)
- 다크 모드, reduce motion 지원
